## How to use

### Option 1 ‚Äî Browse manually
Each category lives in `prompts/<category>.json`. Open the file, find a prompt, copy the `content` field, paste it into your AI tool of choice.

### Option 2 ‚Äî Use it inside YourRender.ai (recommended)
[Sign up for YourRender.ai](https://yourrender.ai) and these prompts (along with thousands more from the broader community) are accessible **in one click** from the platform's Templates panel ‚Äî across all 15+ AI engines, with editable variables, preview images, and one-click generation.

### Option 3 ‚Äî Programmatic access
Each JSON file is structured with `id`, `title`, `content`, `description`, `sourceMedia` (preview images), and `needReferenceImages`. Build your own integration by reading these files directly.

## YourRender.ai's 7-Step Method

Each prompt in this collection is tagged against YourRender.ai's production methodology:

1. **Concept** ‚Äî What story does this visual tell?
2. **Design** ‚Äî Which aesthetic, palette, composition?
3. **Publish** ‚Äî Where will this run (platform, format)?
4. **Empire** ‚Äî How does this fit a recurring brand system?
5. **Studios** ‚Äî What production-grade quality is needed?
6. **Public** ‚Äî How does this serve the audience?
7. **Client Success** ‚Äî Does this drive a measurable outcome?
